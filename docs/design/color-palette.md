Reference the values below when designing new screens, components, or marketing surfaces. Favor dark text on light backgrounds; never use yellow text.

## Core Palette

| Token | Hex | Usage |
| --- | --- | --- |
| `Colors.primary` | `#FFC727` | Brand yellow, primary buttons |
| `Colors.primaryDark` | `#E0A800` | Button hover, emphasis text |
| `Colors.primaryLight` | `#FFE58F` | Subtle highlights |
| `Colors.textPrimary` | `#1A1A1A` | Main copy on light backgrounds |
| `Colors.textSecondary` | `#333333` | Secondary copy, body text |
| `Colors.textMuted` | `#555555` | Captions, helper text |
| `Colors.textOnDark` | `#FFFFFF` | Text on `Colors.darkBackground` |

## Backgrounds & Surfaces

- App background: `#FAFAF7`
- Section background: `#F6F4E8`
- Highlight background: `#FFF4C2`
- Dark background: `#1D1D1D`
- Card surface: `#FFFFFF`
- Elevated card: `#FDFBF4`
- Pressed surface: `#F3EABE`

## Accents & Status

- Accent orange: `#FF8A00`
- Accent warm red: `#E53935`
- Accent deep blue: `#005BBB`
- Success: `#38A169`
- Warning: `#FFB703`
- Error: `#E53935`
- Info: `#2F80ED`

## Buttons

- **Primary:** background `#FFC727`, text `#1A1A1A`
- **Secondary:** background `#FFFFFF`, border `#FFC727`, text `#E0A800`
- **Danger:** background `#E53935`, text `#FFFFFF`

## Borders & Dividers

- Border light: `#E4E4DA`
- Border medium: `#C6C6B8`
- Divider: `#D6D6CE`
